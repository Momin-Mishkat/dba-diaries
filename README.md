# DBA Diaries

Field notes from a working Oracle DBA — performance tuning, backups, high availability, and the production incidents that taught me the most.

**Live site:** https://momin-mishkat.github.io/dba-diaries/

## About

Written by **Momin Mishkat**, Oracle DBA. This is a personal blog — not certification-guide theory, but the actual checks, scripts, and decisions used day to day on production Oracle databases.

Topics covered:
- Performance tuning & AWR analysis
- Backup & recovery (RMAN)
- High availability (Data Guard, RAC)
- Day-to-day production support

## Tech

Single static `index.html` file — no build step, no framework, no server required. Just open it in a browser or host it anywhere that serves static files (GitHub Pages, Netlify, etc).

## Running locally

Clone the repo and open `index.html` directly in any browser:

```bash
git clone https://github.com/momin-mishkat/dba-diaries.git
cd dba-diaries
open index.html   # or just double-click the file
```
